# Knex SQL query builder

- refer-docs:https://knexjs.org/

* Install Dependencies:

```bash

npm install

```

- Start the Server :

``` bash
npm install
```

- Migrate to PostgresSQL

* npx knex migrate:latest --knexfile db/knexfile

```bash
npm run  migrate
```

- Seed to PostgresSQL

* npx knex seed:run latest --knexfile db/knexfile

```bash
npm run seed
```
